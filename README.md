📊 Financial Insights Dashboard (Power BI)
📌 Overview

This project is a Power BI dashboard created to analyze financial performance across multiple branches from FY 2023 to 2025. It helps track revenue, profit, expenses, and key financial metrics in a clear and interactive format.

🎯 Objectives
Track overall financial performance (Revenue, Profit, Expenses)
Compare performance across branches and years
Identify trends in profit and cost
Provide simple reports for better understanding of financial data
🛠️ Tools Used
Power BI
DAX
Data Modeling
📈 Key Features
Dashboard showing Revenue, Gross Profit, EBITDA, and Net Profit
KPI metrics like Gross Profit % and Net Profit %
Branch-wise performance analysis
Monthly profit trend analysis
Profit & Loss statement (Amount and Percentage view)
Balance Sheet overview
Interactive filters for Year and Branch
📊 Insights
Revenue and profit show steady growth from 2023 to 2025
Some branches perform better than others in terms of profit
Expenses like staff cost and commissions take a major share
Profit margins vary across years and branches
